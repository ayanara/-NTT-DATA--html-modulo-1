Introdução ao HTML - Usando tabelas em HTML

Quando se tem um conjunto de dados e precisa-se apresentá-los ordenadamente, uma boa maneira de fazê-lo é usando as tabelas. Elas são basicamente compostas de linhas (tr.../tr) e colunas (td.../td) cruzadas entre si.
